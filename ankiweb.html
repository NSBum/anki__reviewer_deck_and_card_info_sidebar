This is an extended modification of the add-on Extended Card Stats During Review. I made this for myself because I wanted to have support for the night mode add-on and see different info during review: see deck options, see fewer card info, have a shorter and less wide table of repetitions and have some styling. This version makes it easier to configure what is shown in the sidebar (if you know some python). If you you run into problems with this add-on you should use glutanimate's original. Use this version at your own risk. Feedback or improvements (concerning my changes) are welcome. It's best to open an issue in my github repo).
